# Arbitryon

## Setup

1. Clone the repository:
   ```bash
   git clone https://github.com/hasnain3142/arbi-tryon.git
   ```

2. Make the setup script executable:
   ```bash
   chmod +x setup.sh
   ```

3. Run the setup script:
   ```bash
   ./setup.sh
   ```

4. Initialize Conda:
   ```bash
   conda init
   source ~/.bashrc
   ```

5. Activate the Conda environment:
   ```bash
   conda activate idm
   ```

6. Run the server:
   ```bash
   python gradio_demo/app2.py
   ```
